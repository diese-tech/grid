# GRID // ALPHA

GRID // ALPHA is a cinematic spatial web experience for diese-tech projects.

This repository now contains the Next.js foundation plus the planning docs and reference assets that guide future batches.

## Local Development

```bash
npm install
npm run dev
```

Build and lint:

```bash
npm run lint
npm run build
```

## Builder Notes

Start with these docs before adding visual or traversal systems:

- `docs/TECHNICAL_ARCHITECTURE.md`
- `docs/IMPLEMENTATION_SEQUENCE.md`
- `docs/DATA_MODEL.md`
- `docs/DEPLOYMENT_SAFETY.md`
- `docs/BRANCHING_AND_RELEASE_WORKFLOW.md`

Batch 1 intentionally creates only the app foundation and seed data. Cinematic boot, homepage transit, Lab, node map, dossier zoom, audio, and event states belong to later batches.
