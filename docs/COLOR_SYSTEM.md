# Color System

## Purpose

This doc defines how color should behave inside The Grid. Color is not decoration here. It communicates mood, hierarchy, activity, danger, and discovery.

---

# Q/A Summary

1. The dominant palette should feel like a mix between Portal 2, Kamino, and Deep Stone Crypt.

2. Red should be used rarely and intentionally. It represents warnings, unstable systems, raw sectors, and operational danger.

3. The Grid should feel like a cold environment with warm human pockets.

4. Overall brightness should land between readable dark mode and bright atmospheric futurism.

5. Glow should be limited to active or important systems.

6. Navigation should generally avoid glow unless it reveals something special or raw.

7. Status indicators should use pulses and rings instead of noisy dashboards.

8. Charts and telemetry should feel useful first, atmospheric second.

---

# Core Palette Direction

The Grid should feel:

- industrial
- cold
- atmospheric
- operational
- immersive
- lightly dreamlike

The palette should create emotional familiarity and scale.

---

# Primary Colors

## Base Environment

Primary environment tones:

- charcoal
- deep gray
- industrial black
- gunmetal
- muted steel

These tones form the stable shell of The Grid.

---

## Operational Accents

Primary operational accents:

- icy blue
- muted cyan
- cold white
- pale interface glow

These colors represent:

- active systems
- operational stability
- connected sectors
- clean infrastructure

---

# Restricted Accent Colors

## Red

Red is sacred.

Use only for:

- unstable systems
- operational warnings
- dangerous experimentation
- corrupted sectors
- hidden/raw areas
- emergency states

Red should feel meaningful because it appears rarely.

---

# Glow Philosophy

Glow is a communication layer.

Allowed glow targets:

- active nodes
- selected systems
- status indicators
- active panels
- cursor focus
- operational signals
- lightly emphasized buttons

Do not apply glow universally.

The Grid should never look like RGB overload.

---

# Environmental Contrast

The Grid should balance:

- dark atmosphere
- readable interfaces
- soft operational highlights
- restrained contrast

The site should feel immersive without becoming muddy or unreadable.

---

# Warm Human Pockets

Although the world is mostly cold and industrial, some areas can contain warmer human energy.

Examples:

- operator spaces
- archived notes
- humor moments
- active personal projects
- memory-linked sectors

These areas should subtly contrast the colder operational shell.

---

# Forbidden Color Behaviors

Avoid:

- rainbow RGB aesthetics
- neon overload
- crypto-style gradients
- generic cyberpunk palettes
- excessive saturation
- color chaos
- glow spam

Color must feel intentional and authored.

---

# Final Color Goal

The Grid should feel like:

- cold infrastructure
- atmospheric systems
- operational memory
- restrained futurism
- familiar impossible spaces
