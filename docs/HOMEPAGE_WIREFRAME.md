# Homepage Wireframe

## Purpose

The homepage is not "home."

The homepage is the transit layer.

It exists to:

- orient users
- establish atmosphere
- introduce traversal
- imply larger systems
- guide users toward The Lab

---

# Homepage Emotional Goal

The homepage should feel like:

- a quiet futuristic station
- a liminal transit layer
- an operational waiting space
- a connected system hub

Primary emotional targets:

- intrigue
- curiosity
- movement
- subtle unease
- calm operational atmosphere

---

# Core Homepage Layout

## Main Scene

The homepage should primarily be:

> a station platform with route signage.

Possible environmental elements:

- suspended signage
- transparent route maps
- distant sectors
- station lighting
- moving transit lines
- implied unseen areas
- subtle environmental movement

---

## Overlay Structure

The homepage uses overlays sparingly.

Overlays can include:

- current sector
- operator message
- route markers
- active destination indicators
- minimal status text

Avoid:

- giant dashboard panels
- excessive statistics
- full project cards everywhere

---

## Route Signage

Route signage should feel:

- futuristic
- neo-punk
- industrial
- transit-oriented
- functional
- worn but operational

The signage should feel like:

- station infrastructure
- environmental wayfinding
- system navigation

NOT:

- fantasy UI
- generic cyberpunk neon overload

---

## Mini Node Map Placement

Recommended placement:

- lower corner
- transparent
- integrated into station UI

Behavior:

- usable
- clickable
- unlabeled
- exploratory

Users should learn the map naturally.

---

# Homepage Navigation Flow

## Movement Into The Lab

Preferred traversal:

- route sign interaction
- train/transit transition
- environmental movement
- implied sector loading

The transition should feel:

- cinematic
- restrained
- systemic
- spatial

NOT:

- like opening another webpage.

---

# Idle Homepage Behavior

If users idle:

- environment subtly changes
- station audio shifts
- route signs flicker
- announcements distort
- subtle anomalies emerge
- node behavior changes

The homepage should slowly feel:

> alive.

---

# Homepage Success Criteria

Users should leave the homepage remembering:

- the station atmosphere
- the route signage
- traversal
- the minimap
- curiosity toward The Lab
